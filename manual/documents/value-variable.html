# 变量

# 全局变量 Global Variable

一个全局变量的当前值,全局变量即属于游戏本身的变量.

  • 变量 获取此变量的值.

# 玩家变量 Player Variable

一个玩家变量当前的值,玩家变量即属于一个特定玩家的变量.

  • 玩家 获取此玩家的变量值.
  • 变量 获取此变量的值.
